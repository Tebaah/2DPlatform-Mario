# Super Mario Platformer

![Game Screenshot](screenshot.png)

## Description

Super Mario Platformer is a 2D platform game inspired by the classic Mario games. The objective of the game is to guide the main character, Mario, through various levels, overcoming obstacles and enemies to reach the end and rescue Princess Peach.

## Features

- Classic platformer gameplay
- Multiple levels with increasing difficulty
- Power-ups and collectibles
- Different types of enemies and obstacles
